# Incident Response (IR), Business Continuity (BC), and Disaster Recovery (DR) Concepts

## Incident Terminologies

| Terminology | Meaning |
| :--- | :--- |
| Personally Identifiable Information (PII) | Info that can be used to trace individual's identity | 
| Breach | unauthorized disclosure of data |
| Event | anything observable that happen in networks |
| Incident | events that jeopardizes the CIA triad of systems |
| Intrusion | unauthorized user gaining access (the proecss itself) |
| Zero Day | unknown vulnerablities, leaving the developers zero day to fix it before the vulnerablity being exploited |
